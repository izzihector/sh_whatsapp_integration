All in One WhatsApp Integration - Sales, Purchase, Account and CRM

Using this module you can send Quotations, Sale Order, Invoices, Bills, RFQs, Purchase Orders, and direct to Clients/Vendor's WhatsApp. You can easily send PDF of sale, purchase, invoice & inventory documents using URL.

How it works? Click here to watch video

Features

Easy to send sales, purchase, invoice & inventory report URL in WhatsApp.

Easy to send messages to Customer/Vendor/Contacts about Quotation, Sale Order, Delivery Order, Invoice, Bill, Credit Notes, Payment Info, RFQ, Purchase Order, Incoming Order, Lead, Opportunity.

We have 3 way to show message content

A) Order Information in Message - It will Display Only Order Information in Message.

B) Order Product Detail in message - It will display Product Name, Qty, Price, Disc in the message.

C) Signature - It will Display signature at last in Message.

D) Display in Chatter Message - Easy to track send a message which is sent in WhatsApp.

We have given 2 buttons in each form

A) Send By WhatsApp - If you want to change anything in message.

B) Send By WhatsApp Direct - If you don't want to change anything in message.

You can enable disable this easily in sales, purchase, accounting, inventory as we have given separate groups for that.

We have given one quick menu option to send a WhatsApp message to anyone. Users can easily send a message to anyone using that menu.

No, configuration required.

Easy to install and use it.
